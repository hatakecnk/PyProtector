# How To Install
```
$ pkg install git python2
$ git clone https://github.com/hatakecnk/PyProtector
```

# How To Execute
```
$ cd PyProtector
$ python2 enc -i [input] -o [output]
Ex : python2 enc -i in.py -o out.py
```

# For Help
```
$ python2 enc.py -h
```
